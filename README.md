# zs-jenkins
这是一个基于Jenkins的持续集成和持续交付的docker镜像，主要用于构建node、Cordova项目！

# 镜像拉取地址
```shell
$ docker pull shuoer/zs-jenkins
$ docker pull registry.cn-hangzhou.aliyuncs.com/shuoer/zs-jenkins
```
